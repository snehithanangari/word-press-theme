This plugin displays a simple custom notice at the bottom of every WordPress page.

== Description ==
A beginner-friendly plugin to add a site-wide footer notice.

== Installation ==
1. Upload `simple-notice-plugin` folder to `/wp-content/plugins/`
2. Activate the plugin from your WordPress admin panel
